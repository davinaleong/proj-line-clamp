# Line Clamp

A simple demo to showcase how to limit lines of text content.

This page can be found [here](https://davinaleong.github.io/proj-line-clamp/).

## Getting Started

1. Fork this repo
2. Run `npm install`
3. Run `npm run dev`
4. Go to [http://127.0.0.1:5173/](http://127.0.0.1:5173/) to view content.
5. Happy coding!
